# 🐾 Toti - Planos de Saúde Pet

![Next.js](https://img.shields.io/badge/next.js-000000?style=for-the-badge&logo=next.js&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/tailwindcss-0F172A?style=for-the-badge&logo=tailwindcss)
![TypeScript](https://img.shields.io/badge/typescript-%23007ACC.svg?style=for-the-badge&logo=typescript&logoColor=white)

A **Toti** é uma plataforma moderna de planos de saúde para pets, focada em oferecer uma experiência acolhedora e intuitiva para tutores. O projeto une um design vibrante com uma arquitetura de software sólida.

## 🚀 Funcionalidades

- **Home Dinâmica**: Alternância de heros para cães e gatos com animações Lottie.
- **Seleção de Planos**: Tabela de preços comparativa com destaque para o plano mais popular.
- **Formulário de Adesão**: Fluxo de assinatura com validação e feedback visual de sucesso.
- **Responsividade Total**: Experiência otimizada para Desktop, Tablet e Mobile.
- **FAQ Interativo**: Seção de dúvidas com sistema de sanfona (accordion).

## 🛠️ Tecnologias Utilizadas

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Animações**: [LottieFiles](https://lottiefiles.com/) & [Framer Motion](https://www.framer.com/motion/)
- **Ícones**: [React Icons](https://react-icons.github.io/react-icons/)

## 🎨 Design (Direção de Arte)
O projeto utiliza uma paleta de cores baseada em tons de roxo, rosa e verde água, buscando transmitir confiança e modernidade. O uso de ilustrações animadas humaniza a interface, criando uma conexão emocional imediata com o usuário.
